# Go Bag

A packing list app.

## Dependencies

Install with `npm install`.

-   [Webpack](https://webpack.github.io) for `require` system, build pipeline, and dev server
-   [Bootstrap](http://getbootstrap.com)
-   [Handlebars.js](http://handlebarsjs.com)
